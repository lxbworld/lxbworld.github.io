在hexo主题（基于Github pages）博客中插入图片的方法

Posted on 2020-04-21

  1. 设置站点配置_config.yml:将post_asset_folder: false改为post_asset_folder: true

  2. 安装插件:npm install https://github.com/CodeFalling/hexo-asset-image -- save

  3. 运行hexo n "XXXXXX",生成XXXXX.md博文时就会在/source/_posts目录下生成XXXXXX的文件夹，将你想在XXXXX博文中插入的照片放置到这个同名文件夹中即可，图片的命名随意。

  4. 添加图片:在想添加的位置写入![](图片名字.图片格式),例如![](1.png)这种方法生成的图片宽度占了整个页面。

  5. 除了第4步的方法，还可以这样<img src="微信图片_20200416200315.jpg" width="50%" height="50%" title="啊啊啊啊啊" alt="嘻嘻嘻"/>这种方式能够调节图片的大小。

关于evernote（印象笔记）连接不到服务器的问题

Posted on 2020-04-21

其实很简单，evernote（国际版）的账号，要在”evernote”的客户端上登录（win or android）；

印象笔记（国内版）的账号，要在”印象笔记”的客户端登录。

这两款产品的服务器存放位置貌似是不一样的。

用错账号登录客户端，会出现连接不到服务器的问题。

p.s. evernote的图标是，圆白底上有只绿色的大象，而印象笔记的图标是绿底上有只黑色的大象。
